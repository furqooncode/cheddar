# 🧀 Cheddar Luxury

> Nigerian men's luxury streetwear, done right.

Cheddar is a Nigerian men's luxury streetwear platform built for those who carry themselves different. Browse a curated catalog of premium fits rooted in Nigerian identity and culture — from clean refined pieces to bold statement drops that command attention. Pay seamlessly with Paystack, track every order, and enjoy an experience as smooth as the fits you're copping. Fast, clean, and built for the man who doesn't settle.

**Wear Nigerian. Move different.**

## ⚙️ Tech Stack

### Frontend
| Tech | Purpose |
|------|---------|
| React.js | UI Framework |
| Tailwind CSS | Styling |
| TanStack Query | Server state & data fetching |
| Zustand | Client state management |
| Supabase | Auth & Database |

### Backend
| Tech | Purpose |
|------|---------|
| Node.js | Server runtime |
| Resend | Transactional emails |
| Paystack | Payments |

---

## 🚀 Getting Started

```bash
git clone https://github.com/furqncode/cheddar.git
cd cheddar
npm install
npm run dev
```

> Make sure to set up your `.env` file with your Supabase, Paystack, and Resend credentials.
